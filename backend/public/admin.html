🌐 Website FAQ Crawler

Single page or deep crawl · Extracts Q&A pairs · Save directly to your chatbot

🤖 Auto-Crawl Settings

Configure the chatbot embed to auto-index your site's FAQs on page load

💡 How Auto-Crawl works

1️⃣

Embed fires

When your page loads with the chatbot script, it silently calls your backend

2️⃣

Backend crawls

Your server fetches the page HTML and extracts FAQ patterns

3️⃣

FAQs indexed

Extracted Q&A pairs are saved to Supabase (or session-only)

4️⃣

TTL cache

Won't re-crawl the same URL until the TTL window expires

⚙️ Configuration

Enable Auto-Crawl

When enabled, the embedded chatbot will crawl and index FAQs automatically on page load.

Save to database

ON = persist to Supabase permanently. OFF = session-only (FAQs lost on page reload).

Deep link discovery

Discover and crawl internal links found on each base URL (up to Max Pages).

Only crawl FAQ/help paths

Only deep-crawl pages whose path matches /faq|help|support/.

Silent mode

Suppress auto-crawl console logs in the browser. Recommended for production.

Run a test crawl to verify extraction before going live.
